Pseudoelementem :first-line, który obejmuje formatowaniem całą pierwszą linię

p
{
color:navy;
font-size:12pt;
}
p:first-line
{
color:red;
font-size:20pt;
font-weight:bold;
}

pseudoelementy: :after oraz :before. Poza :first-letter oraz :first-line w specyfikacji umieszczono jeszcze dwa inne pseudoelementy: :after oraz :before. Oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony. Poniżej zamieszczam odpowiedni przykład.

p:before
{
content:"TO SĄ ĆWICZENIA";
font-size:15pt;
font-style:italic;
font-weight:bold;
}
p:after
{
content:"AUTOR Król Lew";
font-size:15pt;
font-style:italic;
font-weight:bold;
}

Pseudoelement :first-letter pozwala na formatowanie pierwszej litery w bloku tekstu. Przykładowy arkusz spowoduje powiększenie pierwszej litery do wielkości 100 punktów i nadanie jej koloru czerwoneg

Pseudoelement :first-letter pozwala na formatowanie pierwszej litery w bloku
tekstu.
Przykładowy arkusz spowoduje powiększenie pierwszej litery do wielkości 100 punktów i
nadanie jej koloru czerwoneg

Efektem działania powyższego arkusza stylów będzie powiększona pierwsza litera każdego akapitu występującego w dokumencie XHTML/HTML. Zastosowanie pseudoelement :first-letter z innymi znacznikami. Przykład z nagłówkiem stopnia pierwszego.

H1
{
color:navy;
font-size:30pt;
}
H1:first-letter
{
color:red;
font-size:100pt;
}
